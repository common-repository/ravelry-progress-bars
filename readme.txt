== Description ==

Ravelry widget to display progress bars for your current projects. You will need 
to provide your Ravelry API key. This can be found at http://ravelry.com/help/api.
You will also need to supply your Ravelry username.


== Installation ==

1. Upload `ravelryprogressbars.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Supply your username and api key through the widget settings.
